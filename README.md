## 特性

- 🎨 **主题感知** — 自动适配浅色/深色模式
- 🎯 **零配置** — 默认配置即可生效，开箱即用
- 📦 **兼容 shadcn/ui** — 使用相同的模式和样式约定
- 🗺️ **MapLibre GL 驱动** — 全面访问 MapLibre 强大的地图能力
- 🧩 **可组合** — 使用简单的声明式组件构建复杂地图 UI
- 📍 **标记和弹窗** — 丰富的标记系统，支持弹窗、提示和标签
- 🛤️ **路线绘制** — 在地图上绘制路线和路径
- 🎮 **控件** — 缩放、指北针、定位和全屏控件

## 底图服务条款

本项目使用基于 OpenStreetMap 数据的 [CARTO 底图](https://docs.carto.com/faqs/carto-basemaps)。

- **商业用途**：需要 CARTO 企业许可证。可[申请演示](https://carto.com/request-live-demo)了解定价详情。
- **非商业用途**：在 CARTO 的[底图条款](https://carto.com/legal/bmap)下，CARTO 资助对象可免费使用。
- **替代方案**：您可以切换至 [OpenStreetMap](https://www.openstreetmap.org/) 瓦片或任何兼容 MapLibre 的瓦片提供商（MapTiler、Stadia Maps 等）。
