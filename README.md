# mapcn 中文文档

mapcn React 地图组件库中文文档网站。

基于 Astro + Tailwind CSS 4 + MapLibre GL JS 构建。

## 技术栈

- Astro 7.0.7
- Tailwind CSS 4.3.2
- MapLibre GL JS 5.7.0 (CDN)
- TypeScript 6.0.3

## 开发

```bash
npm install
npx astro dev --port 5182
```

## 构建

```bash
npx astro build
```

## 部署

自动部署到 GitHub Pages，自定义域名: mapcn.ndjp.net
